# Design System

## Colors
- **Primary**: `#13c8ec` (Cyan-500 equivalent)
- **Secondary**: `#0f172a` (Slate-900)
- **Background**: `#f8fafc` (Slate-50)
- **Success**: `#22c55e` (Green-500)
- **Warning**: `#f59e0b` (Amber-500)
- **Error**: `#ef4444` (Red-500)
- **Text Primary**: `#1e293b` (Slate-800)
- **Text Secondary**: `#64748b` (Slate-500)

## Typography
- **Font Family**: 'Manrope', sans-serif
- **Headings**: Bold, Slate-900
- **Body**: Regular, Slate-600

## Components
- **Buttons**: Rounded-lg, Primary bg, White text, Hover: opacity-90
- **Cards**: Rounded-xl, White bg, Shadow-sm, Border Slate-200
- **Inputs**: Rounded-lg, Border Slate-300, Focus: Ring-2 Ring-Primary
- **Modals**: Centered, Backdrop blur-sm, Rounded-xl

## Layouts
- **Dashboard**: Sidebar (Left fixed), Header (Top fixed), Main Content (Scrollable)
- **Public**: Header (Sticky), Hero, Features, Footer

## Shadows
- **Soft**: `0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)`
- **Hover**: `0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1)`
